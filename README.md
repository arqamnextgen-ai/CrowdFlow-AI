# CrowdFlow AI – Smart Event Assistant

## Problem
Large-scale events face crowd congestion, long waiting times, and poor navigation.

## Solution
CrowdFlow AI provides smart recommendations for entry, exit, and services using crowd simulation and optimization logic.

## Features
- Smart route & gate recommendations
- Crowd heat map visualization
- Waiting time estimation
- Multi-option suggestions with scoring

## Tech Stack
HTML, CSS, JavaScript
